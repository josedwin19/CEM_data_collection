A comprehensive Python suite designed to extract, analyze, and report metadata from **EPU Cryo-EM data collection sessions** and automate structure retrieval & statistical analysis using the **RCSB PDB GraphQL API**.

## Overview

This suite provides research teams with automated workflows to:
* Extract collection metadata directly from Thermo Fisher Scientific EPU session folders.
* Generate print-ready, clean PDF acquisition reports summarizing detectors, exposure times, pixel sizes, and accumulated electron doses.
* Fetch structural annotations (molecular weight, resolution, stoichiometry, crystallization method, EM software) for specific PDB entries from the RCSB protein data bank API and output analytical plots.

---

## Suite Features

### 1. Data Collection Extractor (`Data_Collection.py`)
Parses session-level XML/DM files generated by standard EPU session directories:
- **Session Settings**: Extracts session name, software version, phase plate usage, and autoloaders.
- **Detector Stats**: Identifies camera commercial name, dimensions, electron counting status, and pixel sizes.
- **Dose Accumulation**: Computes average total dose per movie, dose per fraction, standard deviation, and range.
- **Optics & Defocus**: Parses nominal voltage, Cs values, nominal magnification, and defocus ranges.

### 2. PDF Reporter (`Metadata_PDF_Reporter.py`)
Reads XML outputs and outputs a high-quality, formatted horizontal A4 PDF report with custom execution summaries and structured data tables.

### 3. PDB Structure Analyzer (`PDB_Analysis.py`)
Automates retrieval of metadata for list of PDB IDs:
- **API Queries**: Connects to the RCSB PDB GraphQL API to fetch titles, dates, resolution, method, MW, and software.
- **Exporting**: Generates structured Excel spreadsheets (all MWs vs structures under 200 kDa).
- **Visualization**: Produces three key figures:
  1. Method distribution (Pie Chart).
  2. Resolution/Method comparison under 200 kDa (Bar Chart).
  3. Binned size distribution of structures determined by Cryo-EM (Annotated Bar Chart).

## Installation & Requirements

Ensure you have **Python 3.8+** installed. Follow these steps to clone, configure, and execute the suite:

### 1. Clone or Download the Suite
```bash
git clone <your-repository-url>
cd Cryo
```

### 2. Create and Activate a Virtual Environment (Recommended)
This isolates the package installations from your global Python environment:
```bash
# On Linux/macOS:
python3 -m venv .venv
source .venv/bin/activate

# On Windows (Command Prompt):
python -m venv .venv
.venv\Scripts\activate.bat

# On Windows (PowerShell):
python -m venv .venv
.venv\Scripts\Activate.ps1
```

### 3. Install Dependencies
Install all package requirements using `pip`:
```bash
pip install -r requirements.txt
```

## Usage Guide

### 1. EPU Metadata Text & JSON Extractor

Parse a session folder containing metadata files like `EpuSession.dm` and fraction XML files.

```bash
# General text output printed to console
python Data_Collection.py /path/to/epu_session_dir

# Export results to custom text or JSON file formats
python Data_Collection.py /path/to/epu_session_dir -o report.txt -f text
python Data_Collection.py /path/to/epu_session_dir -o report.json -f json

# Cap fractions XML parsing to a maximum movie count (speed up processing)
python Data_Collection.py /path/to/epu_session_dir --max-movies 200
```

### 2. EPU Metadata PDF Reporter

Compiles XML metadata inside the session folder directly into a PDF report:

```bash
python Metadata_PDF_Reporter.py
```
*Note: This generates a PDF file in the current directory named `CryoEM_Report_YYYYMMDD_HHMM.pdf`.*

### 3. RCSB PDB Structure Analyzer

Analyzes structural data using PDB IDs extracted from a TSV input file.

```bash
# Analyze using the default TSV file (20260429_0035764_summary.tsv)
python PDB_Analysis.py

# Analyze using a custom TSV file
python PDB_Analysis.py path/to/your_input_file.tsv
```

#### Generated Outputs:
* **`pdb_detailed_analysis_full.xlsx`**: Detailed metadata database of all queried PDB entries.
* **`pdb_analysis_under_200kDa.xlsx`**: Filtered dataset of PDB entries under 200 kDa determined via Cryo-EM/Crystallography.
* **`1_method_distribution.png`**: Pie chart representing method distributions.
* **`2_comparative_methods_under_200kDa.png`**: Counts comparison of structures <200 kDa.
* **`3_cryoem_size_distribution.png`**: Size breakdown bar chart for Cryo-EM entries.

## Dependencies

The suite relies on the following major Python modules:
* **`pandas`** - High-performance data structure manipulation and Excel creation.
* **`requests`** - HTTP client queries for GraphQL PDB APIs.
* **`matplotlib`** and **`seaborn`** - Clean, publication-quality graphics and visualizations.
* **`fpdf`** - Native document layout framework for generating PDF reports.
* **`openpyxl`** - Engine for writing Excel formats.
